PDLP - Product Developer Leadership Program Overview

The purpose of the Product Developer Leadership Program (PDLP) is to mentor and train team members to become proficient in all core technologies, processes and standards utilized at Gensuite. Mentors will advise developers through hands-on training exercises; while simultaneously introducing developers to the products they will support. Graduating team members will then be able to execute on projects and product support with minimal technical guidance on basic to intermediate requests.

Program Areas

Each program area below is a starting point and guide to Gensuite Processes, Architecture and programming concepts. Review with your mentor, explore additional content online (e.g. Adobe CF Docs for example) and execute on the mentioned exercises together with assigned coding exercises from your mentor.

  • Gensuite Processes
  • User Interface Design
  • Client-side Scripting
  • Server-side Scripting
  • Database Programming
  • Platform Architecture

Developers

  • Explore each program area and code examples of each standard introduced to demonstrate understanding of each development standard.
  • Review code with Mentors to receive feedback on best practices or ask questions.
  • Be able to explain processes and architecture to your Mentor.

Mentors

  • Coach developers on coding standards, platform architecture and team processes via a daily touch point.
  • Assign exercises mentioned in PDLP documents and additional coding assignments as needed.
  • Work with Unit and Group Leaders on training strategies and transition into product support.
