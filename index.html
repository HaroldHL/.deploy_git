ColdBox (Coldfusion MVC Framework)

ColdBox is a conventions-based MVC Framework. It provides a set of reusable code and tools that can be used to increase your development productivity, as well as a development standard for working in team environments. ColdBox is comprehensive and modular, which helps address most infrastructure concerns of typical CFML applications. It is composed of several standalone libraries that in turn compose the ColdBox Platform.

What is GPG?

GnuPG is a complete and free implementation of the OpenPGP standard as defined by RFC4880(also known as PGP). GnuPG allows you to encrypt and sign your data and communications; it features a versatile key management system, along with access modules for all kinds of public key directories. GnuPG, also known as GPG, is a command line tool with features for easy integration with other applications.

You can get detailed information on GPG website.

Install on mac terminal

use homebrew to install gpg and check it if it already exists

Generate your own GPG key

A large text will pop up to prompt you for some information like selecting encryption algorithm, fill in your email and password of protecting your private key.

When you get this, it means public and secret key created and signed.

At the same time, You’d better regenerate a “revocation certificate” so that you can request an external public key server to revoke your public key if the key becomes invalid later.

List your key

pub: This is followed by the public key characteristics of the key, including the parameters of the key (the encryption algorithm is rsa, length 2048, generated on what time is it, Signing and Certificating, expired one year later) and the ID of the key.

uid: your personal information

sub: This is followed by the subkey characteristics of the key, in much the same format as the public key part (E for use with Encrypting).

Using GPG link to your Github Account

Use ID to export the public key string for the corresponding GPG key.

Intro XML PATH

A SELECT query returns results as a row set. You can optionally retrieve formal results of a SQL query as XML by specifying the FOR XML clause in the query. The FOR XML clause can be used in top-level queries and in subqueries. The top-level FOR XML clause can be used only in the SELECT statement. In subqueries, FOR XML can be used in the INSERT, UPDATE, and DELETE statements. FOR XML can also be used in assignment statements.

PDLP - Product Developer Leadership Program Overview

The purpose of the Product Developer Leadership Program (PDLP) is to mentor and train team members to become proficient in all core technologies, processes and standards utilized at Gensuite. Mentors will advise developers through hands-on training exercises; while simultaneously introducing developers to the products they will support. Graduating team members will then be able to execute on projects and product support with minimal technical guidance on basic to intermediate requests.

Program Areas

Each program area below is a starting point and guide to Gensuite Processes, Architecture and programming concepts. Review with your mentor, explore additional content online (e.g. Adobe CF Docs for example) and execute on the mentioned exercises together with assigned coding exercises from your mentor.

  • Gensuite Processes
  • User Interface Design
  • Client-side Scripting
  • Server-side Scripting
  • Database Programming
  • Platform Architecture

Developers

  • Explore each program area and code examples of each standard introduced to demonstrate understanding of each development standard.
  • Review code with Mentors to receive feedback on best practices or ask questions.
  • Be able to explain processes and architecture to your Mentor.

Mentors

  • Coach developers on coding standards, platform architecture and team processes via a daily touch point.
  • Assign exercises mentioned in PDLP documents and additional coding assignments as needed.
  • Work with Unit and Group Leaders on training strategies and transition into product support.
